Este repositorio contiene una colección de prácticas y ejercicios desarrollados con Unity y C# durante la unidad enfocada en la creación de videojuegos. El objetivo principal es aplicar los fundamentos de programación en un entorno gráfico interactivo, desarrollando mecánicas de juego, manejo de escenas y manipulación de objetos dentro del motor Unity.

📂 Contenido del repositorio

El repositorio está organizado en diferentes carpetas que representan los elementos clave del desarrollo en Unity:

• Escenas (Scenes)
Contienen los distintos niveles o pantallas del videojuego, incluyendo menús, gameplay y pantallas finales.

• Scripts en C#
Archivos que controlan la lógica del juego, como movimiento de personajes, interacción, físicas y eventos.

• Prefabs
Objetos reutilizables configurados previamente, como personajes, enemigos o elementos del entorno.

• Assets
Recursos utilizados en el proyecto (modelos, texturas, sonidos, animaciones, etc.).

• Archivos de práctica
Ejercicios adicionales para reforzar conceptos específicos dentro de Unity y C#.

🚀 Temas principales abordados

A lo largo de las prácticas se desarrollan proyectos que cubren:

• Uso de C# en Unity (MonoBehaviour)
• Manipulación de GameObjects y componentes
• Creación y gestión de escenas múltiples
• Control de cámara y jugador
• Uso de físicas (Rigidbody, Colliders)
• Detección de colisiones y triggers
• Entrada de usuario (teclado, mouse)
• Creación, lectura y modificación de archivos (persistencia de datos)
• Uso de prefabs y reutilización de objetos
• Organización de proyectos en Unity
• Introducción a interfaces de usuario (UI)

🛠️ Requisitos

Para ejecutar los proyectos necesitas tener instalado:

• Unity Hub
• Editor de Unity (versión compatible con el proyecto)
• .NET SDK
• Visual Studio o Visual Studio Code (con soporte para C#)
• Sistema operativo compatible con Unity

🎯 Objetivo de aprendizaje

El propósito de estas prácticas es que el estudiante:

• Comprenda cómo aplicar C# dentro de un entorno gráfico interactivo
• Desarrolle lógica de programación orientada a videojuegos
• Aprenda a estructurar proyectos en Unity
• Implemente mecánicas básicas de juego
• Maneje múltiples escenas y flujo del juego
• Trabaje con persistencia de datos mediante archivos
